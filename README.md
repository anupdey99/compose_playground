
# Compose Playground

Simple project for playing with jetpack compose library, features etc.


## Explore

- Crypto currency data
- Stock market data
- Custom music volume knob
- Custom progress bar
- Registration form
- Light sensor

## API

- [alphavantage](https://www.alphavantage.co/)
- [coinpaprika](https://coinpaprika.com/)
